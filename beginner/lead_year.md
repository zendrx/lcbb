# Leap Year

## Description
Write a program that reads a year from standard input and determines whether it is a leap year.

## Requirements
Your program must:
1. Read a single integer (the year) from stdin
2. Output "Leap year" if the year is a leap year, or "Not a leap year" if it is not

## Leap Year Rules
A year is a leap year if:
- It is divisible by 4 AND
- It is NOT divisible by 100, unless it is also divisible by 400

## Example Input
2024

## Example Output
Leap year

## Example Input 2
1900

## Example Output 2
Not a leap year

## Example Input 3
2000

## Example Output 3
Leap year

## Notes
- Years are positive integers
- Input will not contain decimals or letters
