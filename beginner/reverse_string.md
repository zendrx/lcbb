# Reverse String

## Description
Write a program that reads a string from standard input and prints the reversed version of that string.

## Requirements
Your program must:
1. Read a single line of input from stdin
2. Output the reversed string (characters in reverse order)
3. Preserve case and any special characters

## Example Input
hello

## Example Output
olleh

## Example Input 2
Crystal

## Example Output 2
latsyrC

## Notes
- Input will not contain newline characters except the final newline
- Empty input should output nothing (just a newline)
- Spaces should be reversed as well
